# Drought Risk Random Forest Classification

## Overview
Random Forest classifier for drought risk mapping using multiple remote sensing indices.

## Model Features
- 100 trees with minLeafPopulation: 3
- Feature importance analysis
- Model validation with confusion matrix
- Drought severity classification (5 classes)

## Input Features
- VCI (Vegetation Condition Index)
- SPI-3 (Standardized Precipitation Index)
- TCI (Temperature Condition Index) 
- LULC (Land Use/Land Cover)
- NDVI (Normalized Difference Vegetation Index)
- DEM (Digital Elevation Model)

## Performance
- Training accuracy: 91%
- Validation accuracy: 85.37%
- Feature importance: SPI-3 (45.65%)
